bloom-highlight

A vanilla Web Component that adds hand-drawn, squiggly highlight effects to any text — no framework required. Inspired by yeunoia/bloom.

How it works

<bloom-highlight> is a custom element that renders an SVG layer behind your text. A combination of feTurbulence and feDisplacementMap filters creates the organic, marker-like wobble. Drop the tag around any inline content and it just works — Shadow DOM keeps styles isolated.

Usage

<script type="module" src="src/index.js"></script>

This is <bloom-highlight>your highlighted text</bloom-highlight> here.

Attributes

type — "box" or "line" (default) for per-line highlights.
scale — Wobble intensity (default 4).
tip — "round" or "square" (default) edge caps.
background-color — Fill color (default #fdfd77).
padding-x / padding-y — Extra space around the text.
gradient — Comma-separated color stops for a gradient fill.
color — Text color (default inherit).
