bloom-highlight

A vanilla Web Component that adds hand-drawn, squiggly highlight effects to any text — no framework required. Inspired by yeunoia/bloom.

How it works

<bloom-highlight> is a custom element that renders an SVG layer behind your text. A combination of feTurbulence and feDisplacementMap filters creates the organic, marker-like wobble. All visual styling is driven by CSS custom properties, so you can define variants with plain CSS classes.

Usage

/* Define your variants in CSS */
bloom-highlight.primary {
  --bloom-bg: #bb66cc66;
  --bloom-scale: 3;
  --bloom-tip: round;
}

/* Then use them in HTML */
<script type="module" src="src/index.js"></script>

This is <bloom-highlight class="primary">highlighted</bloom-highlight> text.

Box mode

In box mode the entire block is wrapped in a single highlight rect, making it ideal for callouts, hero sections, or featured quotes that span multiple lines of text.

Line mode with overlap

With a semi-transparent --bloom-bg, adjacent line rects overlap due to padding, creating a naturally darker color at line junctions. This gives the highlight an organic, marker-on-paper feel where ink pools at the start of each new line. The effect is especially visible when the padding is generous and the fill color has low opacity.

Line mode

Line mode is the default and wraps each line of text individually. This means the highlight bends and breaks naturally across line boundaries, just like a real highlighter pen would on paper. You can mix different variants in the same paragraph to draw attention to distinct ideas. Each variant is a plain CSS class, so they are easy to define and extend.

Gradient variant

Instead of a flat color, you can set --bloom-gradient to a pipe-separated list of color stops for a smooth linear gradient across each line. This works great for playful or decorative highlights.

Marker variant

Crank up the scale and padding for a bold, hand-drawn marker look that stands out on the page. Perfect for headings, annotations, or any text that needs to grab attention.

CSS custom properties

--bloom-bg — Fill color (default #fdfd77).
--bloom-color — Text color (default inherit).
--bloom-scale — Wobble intensity (default 4).
--bloom-tip — "round" or "square" (default) edge caps.
--bloom-padding-x / --bloom-padding-y — Extra space around the text.
--bloom-gradient — Pipe-separated color stops for a gradient fill (e.g. #ff9a9e | #fad0c4).
