bloom-highlight

A vanilla Web Component that adds hand-drawn, squiggly highlight effects to any text — no framework required. Inspired by yeunoia/bloom.

How it works

<bloom-highlight> is a custom element that renders an SVG layer behind your text. A combination of feTurbulence and feDisplacementMap filters creates the organic, marker-like wobble. All visual styling is driven by CSS custom properties, so you can define variants with plain CSS classes.

Usage

/* Define your variants in CSS */
bloom-highlight.primary {
  --bloom-bg: #bb66cc66;
  --bloom-scale: 3;
  --bloom-tip: round;
}

/* Then use them in HTML */
<script type="module" src="src/index.js"></script>

This is <bloom-highlight class="primary">highlighted</bloom-highlight> text.

CSS custom properties

--bloom-bg — Fill color (default #fdfd77).
--bloom-color — Text color (default inherit).
--bloom-scale — Wobble intensity (default 4).
--bloom-tip — "round" or "square" (default) edge caps.
--bloom-padding-x / --bloom-padding-y — Extra space around the text.
--bloom-gradient — Pipe-separated color stops for a gradient fill (e.g. #ff9a9e | #fad0c4).
